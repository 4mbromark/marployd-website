Prenotazione di un set

Solitamente le prenotazioni per un evento aprono due settimane prima, salvo casi particolari, e sono aperte a tutti. Non è possibile prendere accordi prima dell'annuncio ufficiale nelle mie storie di Instagram.

Non ci sono tariffe in quanto scatto sempre gratuitamente, ma i posti sono limitati: verrà data priorità alla scelta dell'orario ai primi che mi scrivono.

Nota che potrei limitare i set di coppia, di gruppi o di alcuni personaggi troppo popolari per motivi creativi. Inoltre, potrei riservare dei set ad esempio per cosplay di personaggi che cerco.

Il set si può annullare in qualsiasi momento senza penalità, purché io venga avvisato.

◆ Durante il set

Durante l'evento, presentarsi puntualmente e già pronti per scattare nella location concordata. Piccoli ritardi sono tollerati, ma dopo un po' potrei decidere di annullare il set in favore di altri.

Non si possono aggregare altre persone se non concordato in fase di prenotazione.

◆ Dopo il set

Le foto verranno consegnate entro un mese dall'evento, salvo diversa indicazione al momento della prenotazione. Un'anteprima sarà disponibile già dopo qualche giorno.

Ricorda che non è possibile editare le mie foto, salvo diversi accordi. Se noti dei difetti di produzione che vorresti correggere, ti invito a segnalarmeli entro una settimana dalla consegna.

Continui solleciti per le foto non aiutano a velocizzarmi, ma ti invito comunque a scrivermi nel caso volessi aggiornamenti sulla consegna.

Le foto puoi pubblicarle a tua discrezione, ma ricordati di taggarmi!
